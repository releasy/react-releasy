# Config

A configuration class.

## Usage

```javascript
import { Config, InMemoryCache, Link } from 'react-releasy';

const config = new Config({
  link: new Link({ url: 'https://yourserveraddress.com/graphql' }),
  cache: new InMemoryCache(),
  devTools: true,
  networkLogger: true,
});
```
